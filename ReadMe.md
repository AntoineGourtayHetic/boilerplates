# Boilerplates

This repo contains a collection of "starter apps": ready-to-run Node.JS
applications for different collections of libraries.

## Includes:

 - express
 - socket.io
 - dnode

## Installation

     git clone git://github.com/nodeapps/boilerplates.git
     cd boilerplates
     ls # See which boilerplates are available!

Each individual boilerplate folder contains its own ReadMe.md.
